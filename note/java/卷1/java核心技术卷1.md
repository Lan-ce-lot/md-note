## 1. 概述

### Java白皮书的关键术语

#### 简单性

Java是C++的一个"纯净"版本。无头文件、指针语法、结构、联合、操作符重载等

#### 面向对象

Java与C++的主要不同点在于多重继承，在Java中用接口。

#### 分布式

例程库处理HTTP和FTP之类的TCP/IP协议

#### 健壮性

1. 早期问题检测
2. 后期动态检测

#### 安全性

1. 运行时堆栈溢出
2. 破坏自己进程外的内存
3. 未经授权读写

#### 体系结构中立

Java虚拟机

#### 可移植性

Java规范中没有依赖具体实现的地方，基本数据类型都有明确说明。

#### 解释型

Java解释器在任何机器上可以直接在移植了解释器的机器上执行Java字节码。
